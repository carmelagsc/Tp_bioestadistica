
Hipótesis y Técnicas Estadísticas para Base de Datos de Enfermedad Renal Crónica (CKD)
---------------------------------------------------------------------------------------

A. Hipótesis con variables categóricas

1. Hipótesis: La presencia de hipertensión (htn) está asociada con CKD.
   - Técnica: Chi-cuadrado de independencia
   - Código en R:
     table(datos$htn, datos$classification)
     chisq.test(datos$htn, datos$classification)

2. Hipótesis: La presencia de diabetes (dm) aumenta el riesgo de CKD.
   - Técnica: Chi-cuadrado o Fisher (si hay frecuencias bajas)
   - Código en R:
     chisq.test(datos$dm, datos$classification)

3. Hipótesis: Los pacientes con apetito pobre (appet) tienen más CKD.
   - Técnica: Chi-cuadrado
   - Código en R:
     chisq.test(datos$appet, datos$classification)


B. Hipótesis con variable numérica y una categórica

4. Hipótesis: Los pacientes con CKD tienen menor hemoglobina (hemo).
   - Técnica: t-test
   - Código en R:
     t.test(hemo ~ classification, data = datos)

5. Hipótesis: Los niveles de creatinina (sc) son más altos en pacientes con CKD.
   - Técnica: t-test
   - Código en R:
     t.test(sc ~ classification, data = datos)

6. Hipótesis: La edad media difiere entre pacientes con y sin CKD.
   - Técnica: t-test
   - Código en R:
     t.test(age ~ classification, data = datos)

   Nota: Antes de aplicar el t-test, verificar supuestos de normalidad y homogeneidad de varianzas.
   Código en R:
     shapiro.test(datos$hemo)
     var.test(hemo ~ classification, data = datos)


C. Hipótesis con dos variables numéricas

7. Hipótesis: La hemoglobina (hemo) y la creatinina (sc) están correlacionadas negativamente.
   - Técnica: Correlación de Pearson o Spearman
   - Código en R:
     cor.test(datos$hemo, datos$sc, method = "pearson")


D. Hipótesis multivariadas (modelos predictivos)

8. Hipótesis: Hemoglobina, creatinina y presión arterial predicen CKD.
   - Técnica: Regresión logística
   - Código en R:
     modelo <- glm(classification ~ hemo + sc + bp, data = datos, family = "binomial")
     summary(modelo)

9. Hipótesis: El apetito pobre y la anemia aumentan la probabilidad de CKD.
   - Técnica: Regresión logística con variables categóricas
   - Código en R:
     modelo <- glm(classification ~ appet + ane, data = datos, family = "binomial")
     summary(modelo)
